SmartSlate Polaris Scrollbar Design

Premium brand-consistent scrollbar with glassmorphism effects and subtle glow. Scroll the content below to see the scrollbar in action.

Interactive Demo

This scrollable area demonstrates the brand scrollbar. Try scrolling, hovering, and dragging the scrollbar thumb to see all states.

Design Features

  • Glassmorphism track with 8px backdrop blur
  • Linear gradient thumb with primary brand color (#a7dadb)
  • Subtle glow effect on hover and active states
  • Smooth 300ms transitions with cubic-bezier easing
  • Touch-friendly 12px width on mobile devices
  • WCAG AA compliant contrast ratios
  • Cross-browser compatible (Chrome, Safari, Edge, Firefox)
  • Respects prefers-reduced-motion user preference

Brand Colors Used

Primary Accent: #a7dadb - Main brand color

Primary Accent Light: #d0edf0 - Hover highlights

Background Paper: #0d1b2a - Track background

Background Dark: #020c1b - Page background

Implementation Details

CSS Class: .scrollbar-brand

Track Width: 10px (desktop), 12px (mobile)

Border Radius: 6px for smooth, rounded appearance

Transition Timing: 300ms cubic-bezier(0.4, 0, 0.2, 1)

Accessibility Considerations

The scrollbar design meets WCAG AA standards:

  • Minimum 8px width exceeds 7.5px standard
  • Touch-friendly 12px width on mobile/tablet
  • Sufficient contrast between thumb and track
  • Clear visual feedback on hover and active states
  • Keyboard navigation unaffected
  • Transitions disabled for reduced-motion preferences

Browser Support

  • Chrome 90+ (Full glassmorphism support)
  • Safari 15.4+ (Full support with webkit prefix)
  • Edge 90+ (Full support)
  • Firefox 97+ (Simplified styling, no glassmorphism)
  • Opera 76+ (Full support)

Usage Example

<div className="scrollbar-brand max-h-[60vh] overflow-y-auto">
  {/* Your scrollable content here */}
</div>
