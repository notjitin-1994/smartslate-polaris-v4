## Color Scheme

### Background Layers

```
┌─────────────────────────────────┐
│ Page Background: #020C1B        │ ← Deep space blue
│   ┌──────────────────────────┐  │
│   │ Glass Container:         │  │
│   │ border-white/10          │  │ ← Subtle glass border
│   │   ┌───────────────────┐  │  │
│   │   │ Marketing: Linear │  │  │
│   │   │ gradient from     │  │  │
│   │   │ #0d1b2a/80 to     │  │  │
│   │   │ #020c1b/60        │  │  │ ← Gradient background
│   │   │                   │  │  │
│   │   │ Auth: bg-white/5  │  │  │ ← Translucent overlay
│   │   └───────────────────┘  │  │
│   └──────────────────────────┘  │
└─────────────────────────────────┘
```

### Accent Colors

- **Primary**: #a7dadb (Teal) - Headlines, icons, buttons
- **Gradient**: Primary 500 → 300 → 500 - Hero text
- **Glow**: rgba(167,218,219,0.12) - Subtle backdrop glow

### Text Hierarchy

- **Hero**: `text-white` at 4xl-6xl size
- **Feature Titles**: `text-white` at sm size, semibold
- **Body Text**: `text-white/70` at xs-base size
- **Secondary**: `text-white/60` at xs-sm size

## Accessibility Map

```
<section> ← aria-label="Platform benefits and features"
  │
  ├─ <h1> ← Main headline (screen reader priority)
  │
  ├─ Feature Cards
  │   ├─ Icon div ← Decorative, no alt text needed
  │   ├─ <h3> ← Feature title
  │   └─ <p> ← Feature description
  │
  ├─ Stats
  │   ├─ <div> ← Animated number
  │   └─ <div> ← Label
  │
  └─ CTA Card
      └─ <p> ← Call-to-action text
```

### Focus Order

```
1. [Skip to content] (implicit)
2. Feature card 1
3. Feature card 2
4. Feature card 3
5. Feature card 4
6. CTA card (if interactive)
7. Email input →
8. Password input →
9. Login button →
10. Google OAuth button →
11. Sign up link →
```
