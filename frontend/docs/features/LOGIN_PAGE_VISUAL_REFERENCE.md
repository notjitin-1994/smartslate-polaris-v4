```
✅ Primary (#a7dadb):
   - CTA button gradient
   - Focus glows
   - Status badge
   - Gradient text
   - Link hover states

✅ Secondary (#4f46e5):
   - Card outer glow
   - Complementary accents

✅ White Opacity Scale:
   - 100%: Headings
   - 80%: Labels
   - 70%: Subheadings
   - 60%: Secondary text
   - 50%: Legal links
   - 40%: Placeholders
   - 10%: Borders
   - 5-3%: Backgrounds

✅ Semantic Colors:
   - Green (emerald-400): Success states
   - Red (red-400): Error states
   - Indigo (indigo-400): Info states

✅ Never Used:
   - Random hex codes
   - Hardcoded RGB values
   - Non-brand colors
```

## Performance Optimization

### GPU-Accelerated Properties

```
✅ GOOD (60fps):
   - transform: translateX(), translateY(), scale()
   - opacity: 0 → 1
   - filter: blur() (sparingly)

❌ AVOID (layout thrashing):
   - width, height (dynamic)
   - top, left, right, bottom (animation)
   - margin, padding (animation)
```

### Render Optimization

```
Component Tree:
LoginPageClient (Server Component)
├─ SwirlBackground (Client)
├─ LoginMarketingSection (Client)
│  └─ Framer Motion animations
└─ LoginFormContent (Client)
   ├─ AuthInput (Client)
   │  └─ useState (focused, validation)
   ├─ PasswordInput (Client)
   │  └─ useState (visible, focused)
   └─ GoogleOAuthButton (Client)

Total Client Components: 6
Total State Hooks: 4 (minimal rerenders)
```
