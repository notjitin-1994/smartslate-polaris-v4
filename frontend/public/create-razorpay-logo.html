Create Razorpay Logo

This tool will create a 256x256px logo with a black circular background and the swirl icon.

  1. The canvas below shows the preview
  2. Click "Download Logo" to save the image
  3. Save it as /public/razorpay-logo.png
  4. Or upload to a CDN and set the URL in your .env.local
