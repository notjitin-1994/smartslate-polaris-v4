## Core Workflows

### Workflow 1: Create & Generate Blueprint (30-45 min)
- Complete static questionnaire (5-10 min)
- AI generates contextual dynamic questions
- Complete dynamic questionnaire (15-30 min)
- AI generates comprehensive blueprint (10-30 sec)
- Blueprint ready for review

### Workflow 2: Share with Stakeholders (5 min)
- Generate public shareable link
- Send to stakeholders (no auth needed)
- Stakeholders review full blueprint
- Collect feedback
- Update and reshare

### Workflow 3: Present to Leadership (10 min)
- Launch presentation mode
- Navigate slides with keyboard
- Answer questions from dashboard data
- Fullscreen for projection
- Print if needed

### Workflow 4: Export for Documentation (5 min)

- Configure options
- Download file
- Use in workflow

- Export

## Technology Stack

**Frontend**: Next.js 15, React 19, TypeScript 5.7, Tailwind CSS v4
**Backend**: Supabase PostgreSQL, RLS policies
**AI**: Claude Sonnet 4.5 (primary) → Sonnet 4 (fallback) → Ollama (self-hosted)
**State**: Zustand + TanStack Query
**Forms**: React Hook Form + Zod validation
**Animations**: Framer Motion
**Payments**: Razorpay
