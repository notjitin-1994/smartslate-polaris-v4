WORKFLOW EXAMPLES:
================================================================================

WORKFLOW 1: CREATE BLUEPRINT (30-45 minutes)
  1. Complete Static Questionnaire (5-10 min)
  2. AI Generates Dynamic Questions
  3. Complete Dynamic Questionnaire (15-30 min)
  4. AI Generates Blueprint (10-30 sec)
  5. Blueprint Ready for Review

WORKFLOW 2: SHARE WITH STAKEHOLDERS (5 minutes)
  1. Click Share → Get unique link
  2. Send to stakeholders (no auth needed)
  3. Stakeholders view full blueprint
  4. Stakeholders review and provide feedback
  5. Owner updates and reshares

WORKFLOW 3: PRESENT TO LEADERSHIP (10 minutes)
  1. Click "Present" → Launches slide deck
  2. Navigate with keyboard or buttons
  3. Fullscreen for projection
  4. Answer questions from data
  5. Print deck if needed

WORKFLOW 4: EXPORT FOR DOCUMENTATION (5 minutes)
  1. Select export format (PDF, Markdown, JSON, CSV, PNG, JPG)
  2. Configure options if needed
  3. Download file
  4. Use in your workflow

WORKFLOW 5: CREATE CUSTOM REPORT (10 minutes)
  1. Open Report Builder
  2. Select sections to include
  3. Drag to reorder
  4. Customize theme (colors, fonts, spacing)
  5. Preview and export
