# Features Documentation

This directory contains documentation for all major features and systems in Polaris v3.

## Directory Structure

### `/admin` - Admin Dashboard & Tools
Documentation for the admin dashboard, user management, monitoring, and administrative features.
- Admin dashboard implementation and architecture
- Cost monitoring and analytics
- User management and access control
- System diagnostics and health checks
- Alerts and notification systems

### `/auth` - Authentication & Authorization
Documentation for user authentication, roles, and subscription tiers.
- User authentication implementation
- Role-based access control (RBAC)
- Subscription tiers and limits
- User profile management
- Developer access and special roles

### `/limits` - Usage Limits & Tier Management
Documentation for usage limits, tier restrictions, and upgrade modals.
- Tier-based limit enforcement
- Upgrade modal implementations
- Limit checking and blocking
- Rollover system
- Frontend integration

### `/logging` - Logging System
Documentation for the comprehensive logging system.
- Structured logging implementation
- Log aggregation and storage
- Performance monitoring through logs

### `/notifications` - Notification System
Documentation for the notification and alert system.
- Notification architecture
- Alert types and triggers
- User notification preferences

### `/payments` - Payment Integration
Documentation for Razorpay payment integration and checkout flows.
- Razorpay integration setup
- Checkout flow implementation
- Payment webhook handling
- Testing and verification
- Production deployment checklists

### `/ui` - UI Features & Components
Documentation for major UI features and user-facing functionality.
- Mobile responsiveness implementation
- Avatar upload and management
- Settings page revamp
- Profile page features
- Share page implementation
- Modal components
- Presentation mode
- Pricing page design
- Impact metrics redesign
- Touch device optimizations

### `user-feedback-system-architecture.md`
High-level architecture for the user feedback collection system.
